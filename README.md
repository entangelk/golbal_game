# 신규 게임 개발을 위한 게임 트랜드 분석

## 1. 개요

```
현재 게임 회사인 000회사는 신규 게임을 개발하기 위한 초기 단계에 있다. 하지만 세일즈 통로와 규모가 작아 글로벌적인 판매는 어렵다고 판단되어 지엽적인 성공을 위한 게임을 개발하고 싶다.
비용의 문제로 분석에 대한 리소스 투입은 최소한으로 하여 간단한 분석만을 진행하고자 한다.
되도록이면 모험을 하지 않고 보장된 게임을 개발하고자 전체적인 트렌드 분석을 진행하고자 한다. 
```

## 1-1 선결론
| 장르         | 지역    | 퀄리티 | 이유                                                                                     |
|--------------|---------|--------|------------------------------------------------------------------------------------------|
| 슈팅         | 글로벌  | A      | 판매량이 적어지고 있음을 감안, 트리플A에 가지 않고 A정도의 가성비적 개발을 통한 시장 진출 전략 |
| 비주얼 노벨  | 일본    | A      | 일본 내에서 효과적인 판매 가능, 단 일본 시장의 크기에서 매력적이지 않기 때문에 가성비 개발  |
| 보드게임     | 글로벌  | B      | 하락하고 있는 추세의 게임으로 한시적으로 유럽/아프리카 시장에 진입할 수 있을 것으로 예상    |
| 스포츠       | 글로벌  | AAA    | 제대로 된 하나의 스포츠 프랜차이즈는 대체가 없기 때문에 독과점 효과를 누릴 수 있음. 단, 실제 스포츠 게임을 만들기는 매우 어렵기 때문에 판타지적, 전략적, 무관심 스포츠의 게임 개발이 유효할 것으로 예상 |
| 교육/샌드박스| 글로벌 | AA | 샌드박스의 특성상 고퀄리티 게임의 제작은 불가. 다만, 만들게 되면 트렌드의 영향으로 어느정도의 결과물을 획득할 수 있을 것으로 예상 |

## 3. 분석 전략

- 본 분석의 목표는 트렌드 분석에 있다. 하여 기울기를 이용한 시계열 트렌드 분석통해 시장에 유효한 트렌드를 발견한다.
- 부가적으로 지역, 장르, 퍼블리셔 별 세부 분석을 진행한다.

- 그러나 연도별 발매 수와 판매량의 그래프가 유사함을 기준으로 발매수의 차이로 인해 판매량에 매우 큰 영향을 주고 있다. 즉, 게임의 판매는 발매년도를 기준으로 시간이 지날수록 급격하게 하락한다는 것을 알 수 있다.

## 6. 결론
```
1. 종합
게임은 발매 후 판매량이 급격하게 하락하여 소위 '오픈런'이 매우 중요함
최근 전 세계적으로 판매량이 떨어지고 있어 게임 시장은 박리 다매의 전략이 점점 어려워질 수 있음

판매량의 경우 하나의 대표적 게임의 시리즈가 계속해서 판매되는 것을 확인할 수 있음 이는 위의 박리 다매전략에 대한 분석과 일맥 상통하는 결과

2. 지역별
북미 지역에서는 하락세가 줄어들어 변곡점을 향하고 있는 추세임 (상승 가능성)
일본은 타 지역과 다르게 매니악한 판매량을 보이고 있음
유럽/아프리카 지역은 게임의 판매량이 줄어들 고 있음

3. 장르별

슈팅/스포츠의 경우 대체 게임이 없어 장르적 특수성에 의해 판매량을 지지할 가능성이 있음
단, 슈팅의 경우 판매량의 추세가 매우 급하게 하락하고 있는 추세임.
비쥬얼 노벨의 경우 판매량은 낮으나 크리틱 스코어가 매우 후함 -> 일본 한정 매니아들을 모집하여 충성고객화 가능성
보드 게임의 경우 유럽/아프리카에서 게임 내 비율의 상승을 보이고 있음 -> 한정적 시장에 유효할 것으로 예상

4. 기타
통계적 검정에 의하면 유명 퍼블리셔가 참여하는 것이 판매량에 더욱 긍적적인 효과를 주고 있음
```
